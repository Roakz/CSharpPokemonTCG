# PokemonTCGBattle

Welcome to the Pokemon TCG Console Game! This project is a console-based game where players are dealt Pokemon cards at random to battle against the computer.

## Getting Started

These instructions will help you set up and run the project on your local machine.

### Prerequisites

- [.NET Framework 4.8](https://dotnet.microsoft.com/download/dotnet-framework/net48)
- [Visual Studio 2022](https://visualstudio.microsoft.com/vs/)

### Installation

1. Clone the repository:
  git clone https://github.com/yourusername/PokemonTCGBattle.git
2. Open the solution in Visual Studio 2022.
3. Ensure the JSON file `PokemonCardData.json` is placed in the correct directory:

### Usage

1. Build the solution in Visual Studio.

2. Run the project. The console will display the game menu.

3. Follow the on-screen instructions to play the game:
    - Press any key to begin.
    - You will be dealt 5 Pokemon cards at random.
    - The battle will begin against the computer.
